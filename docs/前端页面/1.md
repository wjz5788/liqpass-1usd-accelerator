## 1) Figma 级布局规格（可直接照抄）

### 画板与栅格

* Desktop：`1440px` 宽
* 内容最大宽：`1200px`
* 页面左右 padding：`24px`
* 栅格：`12 columns`，gutter `16px`
* 基础圆角：`16px`
* 卡片内边距：`16px`（密集区 `12px`）
* 区块间距：`16px`（大区块 `24px`）

### 层级结构（从上到下）

#### A) Header（固定在内容区顶部）

高度：自适应（建议 72-96）

* 第一行：返回、项目名、verified、tags、share
* 第二行：description（最多 2 行）
* 第三行：links（Demo/Repo/Docs/X/Video）

**组件**：`ProjectHeader`

---

#### B) 顶部双栏：Radar + Window（强聚焦区）

布局：`8/4` 分栏（Desktop）

* 左：`ProjectRadarPanel`（多事件 mini bars）
* 右：`WindowControlPanel`（倒计时、阶段、系统角色、窗口按钮）

高度：建议 160-200

---

#### C) Opportunity Strip（赚钱点提示条）

单行横向卡片（Desktop 4 个槽位）

* Highest Vol
* Thin Book
* New Evidence
* Index Mispricing（可选）

高度：72-96
点击后滚动定位并高亮对应 EventCard（3 秒）。

---

#### D) Event Grid（核心区）

Desktop：`2 columns`（卡片宽大更好读）
Tablet：`1 column`
卡片高度：240-320（可变）

每张 EventCard 结构固定：

1. 顶栏：icon+事件名 + phase/status badge
2. 价格：YES/NO + 24h change + thin/depth chip
3. 证据进度：progress + last evidence time + 提交按钮
4. deadline 倒计时
5. 操作区：Buy YES / Buy NO / Place Order / View
6. resolver 简版（头像/数量）

---

#### E) 双栏：EvidenceFeed + Activity（证据/交易流）

Desktop：`7/5` 分栏
高度：建议 420-520（可滚动容器）

* 左 Evidence：像 PR feed（Pending/Accepted/Rejected）
* 右 Activity：成交、挂单、巨鲸提示

---

#### F) Index/Basket Panel（底部大卡片）

全宽 1 张卡：

* Index Price
* Constituents list
* Mint/Redeem（按钮）
* Deviation hint（可选）

---

#### G) Footer（规则与可信度）

* Resolver/Attestors、Fees、Risk、FAQ（折叠）

---

### 响应式规则（务实）

* `<1024px`：Radar+Window 变上下排列
* `<768px`：Opportunity 变 2x2；EventGrid 1 列；Evidence/Activity 变上下

/** ----------------------------
 * Helpers
 * ---------------------------- */
function clamp01(n: number) {
  return Math.max(0, Math.min(1, n));
}
function pct(n: number) {
  const v = Math.round(n * 1000) / 10;
  return `${v > 0 ? "+" : ""}${v}%`;
}
function formatShortTime(iso: string) {
  try {
    const d = new Date(iso);
    return d.toLocaleString(undefined, { month: "2-digit", day: "2-digit", hour: "2-digit", minute: "2-digit" });
  } catch {
    return iso;
  }
}
function msUntil(iso: string) {
  return new Date(iso).getTime() - Date.now();
}
function humanCountdown(ms: number) {
  if (ms <= 0) return "Expired";
  const s = Math.floor(ms / 1000);
  const d = Math.floor(s / 86400);
  const h = Math.floor((s % 86400) / 3600);
  const m = Math.floor((s % 3600) / 60);
  if (d > 0) return `${d}d ${h}h`;
  if (h > 0) return `${h}h ${m}m`;
  return `${m}m`;
}

function OpportunityStrip({
  highestVol,
  thinBook,
  latestEvidence,
  indexMispricing,
  onJump,
}: {
  highestVol: EventItem;
  thinBook?: EventItem;
  latestEvidence: EventItem;
  indexMispricing: { enabled: boolean; text: string };
  onJump: (eventId: string) => void;
}) {
  return (
    <div className="grid grid-cols-1 md:grid-cols-4 gap-3">
      <OppCard
        title="Highest Vol"
        detail={`${highestVol.topic} (${pct(highestVol.change24h)})`}
        onClick={() => onJump(highestVol.id)}

      <OppCard
        title="Thin Book"
        detail={thinBook ? `${thinBook.topic} (${thinBook.orderbook})` : "No thin books"}

        disabled={!thinBook}

      <OppCard
        title="New Evidence"
        detail={`${latestEvidence.topic} (${formatShortTime(latestEvidence.evidence.lastAt)})`}
        onClick={() => onJump(latestEvidence.id)}
      />
      <OppCard
        title="Index"
        detail={indexMispricing.enabled ? indexMispricing.text : "No deviation"}
        onClick={() => {}}
        disabled={!indexMispricing.enabled}
      />
    </div>
  );
}

function OppCard({ title, detail, onClick, disabled }: { title: string; detail: string; onClick: () => void; disabled?: boolean }) {
  return (
    <button
      onClick={onClick}
      disabled={disabled}
      className={[
        "text-left rounded-2xl border shadow-sm px-4 py-3 transition",

          ? "border-neutral-800 bg-neutral-900/30 text-neutral-500 cursor-not-allowed"
          : "border-neutral-800 bg-neutral-900/50 hover:bg-neutral-900 text-neutral-100",
      ].join(" ")}
    >
      <div className="text-xs text-neutral-400">{title}</div>
      <div className="mt-1 text-sm font-medium">{detail}</div>
    </button>
  );
}
