LiqPass / 1USD Accelerator
GitHub：wjz5788/liqpass-1usd-accelerator

一、项目是什么（给评委 / 外部一眼看懂）

LiqPass / 1USD Accelerator 是一个以「1 美元预测市场」为核心的 Web3 市场基础设施项目,其中还有比赛 / 策略 / 保险模块，其中保险模块已经获得op 资助。

项目通过极低门槛的预测市场机制，把用户的注意力转化为价格信号、市场趋势、项目曝光和早期开发经费，为 Web3 项目、开发者和市场参与者提供一个公开、可量化、可验证的“预赛场”。

该平台可作为 黑客松、比赛、融资和市场决策之前的前置市场，让项目在正式进入下一阶段之前，先接受真实用户参与和市场判断的检验。

二、核心创新：1 美元预测市场（1USD Accelerator）
1. 为什么是 1 美元？

Web3 世界中信息极度泛滥，但大多数“注意力”是无成本、无责任的。
本项目通过 1 美元的最小经济门槛，让用户在表达观点时承担极低但真实的成本，从而：

过滤无效噪声

让判断变得“有价格”

让注意力真正推动市场变化

2. 机制来源与改进

1USD 市场融合了三种成熟机制并做针对性改造：

Pump 的低门槛与强参与感

Polymarket 的“价格即观点”表达方式

LMSR / 变体成本函数（论文讨论核心），用于：

早期：小资金也能推动价格变化，鼓励探索

后期：提高操纵成本，让信号更稳定

其目标不是博彩，而是项目与趋势的市场化发现机制。

三、这个市场同时服务的四类角色
1）普通用户

用 1 美元参与判断，而不是只刷信息

判断正确即可获得回报

注意力第一次真正“有收益 / 有损失”

2）开发者 / 项目方

获得真实曝光，而非营销刷量

在极早期获得一部分开发经费

用市场数据证明“是否有人真正关心这个项目”

3）市场整体

最近的热点、叙事、行情被量化成价格与热度

噪声被交易成本自然过滤

减少 KOL 话语权垄断，增强市场透明度

4）后续投资者 / 黑客松 / 比赛主办方

把这里当作公开的预赛场

在正式比赛或投资前：

看哪些项目被持续关注

看哪些项目有真实参与深度

四、为什么仓库里还有比赛 / 策略 / 保险模块？
重要说明

比赛区、策略区、保险区并不是同一条产品漏斗，也不是围绕量化交易的单一闭环，而是为不同人群设计的并行模块。

它们的存在是为了扩展生态覆盖面，而不是替代 1 美元市场的核心地位。

1）比赛区（Competition）

为策略、带单、AI 机器人提供统一规则下的实盘验证场

所有成绩可对比、可回放、可审计

目的：

让真正稳定赚钱的策略与老师被看见

为后续 AI 量化 / AI 执行提供高质量验证数据

减少“嘴炮带单”欺诈

2）策略区（Strategy）

为不具备深度研究能力的用户提供可复用的赚钱方法

策略可以是：

参数化规则

模板化方法

可被 AI 或人工执行的范式

核心目标：降低参与门槛，提高市场规范度

3）保险区（LiqPass）

面向风险厌恶型用户

提供短周期、规则明确的风险保护工具

让：

不敢碰合约

不敢上杠杆
的用户也能参与市场

保险区的作用是扩人群，而不是提高交易频率。

五、我们不纠结“它是什么”，只看它解决了什么

这个项目不试图定义自己属于哪一类产品，也不执着于“市场基础设施 / 交易工具 / 预测市场”等标签。

唯一判断标准只有一个：

它是否真实满足市场需求，是否让用户赚到钱，是否让参与者持续留下来。

如果市场需要的是：

更快发现有潜力的项目

更清晰判断当前行情和趋势

更公平地让普通用户参与并获利

更透明地让好项目、好策略、好开发者被看见

那这个项目就围绕这些需求不断调整形态。

六、总结（一段话版本）

LiqPass（1USD Accelerator）是一个以 1 美元预测市场为核心的 Web3 市场基础设施项目。

它通过低门槛预测市场，把用户注意力转化为价格、趋势和项目曝光，为早期项目提供市场化的发现与加速机制。

平台同时提供比赛、策略与保险等模块，分别服务能力验证、工具化参与和风险控制等不同人群，并可作为黑客松、比赛和融资之前的公开预赛场。  ，翻译为英文
